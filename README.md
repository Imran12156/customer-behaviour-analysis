# customer_behaviour_analysis
📊 Data Analyst Project
🔍 Overview

This project demonstrates an end-to-end data analysis workflow, starting from data loading to delivering business insights through dashboards and reports.
It showcases skills in data cleaning, exploratory data analysis (EDA), SQL querying, and data visualization.

📁 Dataset
The dataset contains customer/business-related data used for analysis
Includes fields such as customer details, transactions, and performance metrics
Data was initially loaded and processed using Python
📊 Dashboard
An interactive dashboard was built using Power BI
Key features:
KPI tracking (sales, revenue, customer metrics)
Filters and slicers for dynamic analysis
Visual representation of trends and patterns
🛠️ Tools & Technologies
Python → Data loading, EDA, and preprocessing
SQL (PostgreSQL / MySQL / SQL Server) → Data querying and analysis
Excel → Initial data cleaning (if applicable)
Power BI → Dashboard creation and visualization
Gamma → Presentation (PPT) creation
⚙️ Project Steps
1. Data Loading
Imported dataset using Python (Pandas)
Verified structure and data types
2. Data Cleaning
Handled missing values
Removed duplicates
Standardized formats and corrected inconsistencies
3. Exploratory Data Analysis (EDA)
Analyzed data distributions and relationships
Identified trends, patterns, and outliers
4. SQL Analysis
Loaded cleaned data into a relational database
Wrote SQL queries to answer business questions:
Customer segmentation
Revenue analysis
Trend analysis
5. Dashboard Creation
Built interactive visuals in Power BI
Designed KPIs and business metrics
6. Reporting & Presentation
Created a structured report of findings
Designed a presentation using Gamma to communicate insights
🚀 How to Run
Python (EDA & Cleaning)

Install required libraries:

pip install pandas numpy matplotlib seaborn

Run the Python scripts/notebooks:

python eda_analysis.py
SQL
Import dataset into your database (PostgreSQL/MySQL/SQL Server)
Run SQL scripts from the /sql folder
Power BI
Open the .pbix file in Power BI Desktop
Refresh data if needed
Presentation
Open the presentation file created using Gamma (exported as PPT/PDF)
